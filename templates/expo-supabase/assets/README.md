# Assets

The `assets/images/` folder expects these files for the app to build:

- `icon.png` — 1024×1024 app icon
- `splash-icon.png` — used on splash screen and in the auth screen logo
- `android-icon-foreground.png` — 432×432 adaptive icon foreground
- `android-icon-background.png` — 432×432 adaptive icon background
- `favicon.png` — 48×48 web favicon (if building for web)

You can drop in any PNGs that match these dimensions to get started.
For a quick placeholder, any square PNG will work for the first three.
